Документ MVP

Политика конфиденциальности

Эта политика описывает базовые правила обработки данных на сайте Резюме 50+. Перед запуском владелец продукта должен адаптировать текст под юридическое лицо, реквизиты и фактические процессы.

Какие данные собираются

Имя, телефон, выбранный способ связи, комментарий к заявке, страница и источник отправки формы.

Для чего используются данные

Данные используются только для связи по заявке, проведения бесплатного созвона и подготовки предложения по услуге.

Передача данных

При настроенных переменных окружения заявка отправляется в Telegram-чат владельца продукта. Данные не продаются и не передаются третьим лицам для рекламы.

Хранение

Сайт не сохраняет заявки в базу данных в рамках MVP. Фактическое хранение зависит от подключенного Telegram-чата и процессов владельца.

Контакты

По вопросам обработки данных можно написать на hello@resume50plus.ru.
